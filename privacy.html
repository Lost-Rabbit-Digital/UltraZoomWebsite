Privacy Policy

Last updated: April 2025

Data Collection

Ultra Zoom does not collect, store, or transmit any personal data. The extension operates entirely within your browser.

License Validation

If you purchase a Pro subscription, the extension sends a license key to our validation server (api.ultrazoom.app) to verify your subscription status. No personal browsing data is included in this request. The server is hosted on Cloudflare Workers.

Third-Party Services

Pro subscriptions are processed through Paddle, our payment processor. Paddle handles all billing information. We do not have access to your full payment details.

Permissions

Ultra Zoom requests host permissions to inject content scripts that detect images on web pages. These scripts run locally in your browser and do not send page content to any external server.

Open Source

Ultra Zoom is open source. You can review the complete source code on GitHub.

Contact

Questions about this policy? Reach us via GitHub Issues.
