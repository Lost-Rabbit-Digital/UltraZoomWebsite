Getting Started

Ultra Zoom adds instant hover-to-zoom to images across 60+ websites. Follow these steps to get set up:

  1. 1
    Install the extension

    Add Ultra Zoom from the Chrome Web Store or Firefox Add-ons.

  2. 2
    Visit any supported site

    Go to Google Images, Amazon, Reddit, Twitter/X, Instagram, YouTube, or any of the 60+ supported websites.

  3. 3
    Hover over an image

    Move your mouse over any image thumbnail. The full-size version appears instantly in an overlay.

  4. 4
    Dismiss the overlay

    Move your mouse away or press Esc to close the zoomed view.

Keyboard Shortcuts

Ultra Zoom is built for power users. These shortcuts work while an image overlay is active:

Shortcut Action
Esc Close the current zoom overlay
Scroll Up / Down Zoom in or out on the displayed image
← / → Navigate to the previous or next image in a gallery
D Download the current image Pro

Per-Site Control

You decide where Ultra Zoom is active:

  • Master toggle — click the Ultra Zoom toolbar icon and flip the main switch to enable or disable the extension everywhere.
  • Per-site toggle — while on any website, click the toolbar icon and use the site-specific switch to disable Ultra Zoom for that domain only.
  • Global mode — enable "All sites" in the popup to activate Ultra Zoom on websites that don't have a dedicated plugin.

Troubleshooting

Ultra Zoom isn't working on a site I expect it to support

Click the toolbar icon and make sure the extension is enabled both globally and for that specific site. If the site isn't in the supported list, try enabling "All sites" mode in the popup.

The zoom overlay flickers or disappears immediately

This can happen on sites with aggressive hover styles. Try moving your mouse slowly and steadily over the image rather than quickly. If the issue persists on a specific site, let us know and we'll investigate.

The extension toolbar icon is missing

Click the puzzle-piece icon in Chrome or the extensions menu in Firefox to find Ultra Zoom and pin it to your toolbar.

Ultra Zoom conflicts with another extension

Try disabling other extensions one at a time to identify the conflict. Common conflicts occur with other image-handling or zoom extensions. Report the conflict so we can investigate a fix.

My Pro license isn't being recognized

Make sure you're logged in with the same account used during purchase. If the problem continues, contact us at boden@lostrabbitdigital.com with your order details.

FAQ

Is Ultra Zoom free?

Yes. Core hover-to-zoom across 60+ sites is completely free. Pro features (video preview, gallery nav, download, captions) are available for $12.99 one-time or $3.99/month.

Does Ultra Zoom collect my data?

No. Ultra Zoom does not collect, store, or transmit any personal data or browsing history. It operates entirely within your browser. See our Privacy Policy for full details.

Is Ultra Zoom open source?

Yes. Ultra Zoom is open source. You can review the code to verify it does exactly what it claims.

Which browsers are supported?

Ultra Zoom works on Chrome, Edge, Brave, and other Chromium-based browsers via the Chrome Web Store, and on Firefox via Firefox Add-ons.

How do I request support for a new website?

Send us the site URL at boden@lostrabbitdigital.com and we'll look into adding a plugin for it.

Can I use Ultra Zoom on sites not in the supported list?

Yes. Enable "All sites" mode in the extension popup to activate basic hover-to-zoom on any website. Results may vary depending on how the site structures its images.
